## Sobre
Mini MUD textual em Python que demonstra o algoritmo 0/1 Knapsack aplicado a um turno de combate. Cloud enfrenta Sephiroth em um turno crítico. Cloud possui pouca vida, Sephiroth tem 21 pontos de vida restantes, e Cloud precisa escolher habilidades ofensivas usando a mana disponível. O projeto mostra como o 0/1 Knapsack pode otimizar essa escolha em uma lista dinâmica de golpes.

O algoritmo não é tratado como magia ou habilidade interna do jogo. Ele aparece como uma demonstração didática para analisar decisões de combate.
